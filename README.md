## About This Talk

From vibe coding at the terminal to AI agents that run your digital life — this is a practical journey through the tools, patterns, and philosophy behind full-stack personal automation.

### What We'll Cover

| Part | Topic | Description |
|------|-------|-------------|
| **01** | Vibe Coding | Terminal-first philosophy, the AI CLI landscape, and how to build software faster by staying in flow |
| **02** | Agent Setup | Oh My OpenCode plugin system — turning a coding assistant into a powerful multi-tool agent with skills |
| **03** | The Gap | Why coding tools stop at code — and what's missing when you want to automate your entire digital life |
| **04** | Life Automation | A real system that manages tasks, calendar, and servers via Telegram and natural language |

## The Core Pattern

```
natural language → AI agent → action
```

This talk demonstrates how the same pattern that revolutionized coding can now automate your entire digital life.

---

## 5 Lessons

1. **The terminal is the best UI** — Every tool you build, wrap in a CLI first. Scriptable, composable, automatable.

2. **Language is the interface** — Natural language commands beat menus, forms, and dashboards.

3. **Integration beats features** — One connected system that does 80% is worth more than 10 perfect apps that don't talk.

4. **Automation compounds** — 10 minutes of setup saves 10 minutes every week. Stack 20 automations.

5. **AI has conquered coding — now it should conquer life** — OpenCode + gws CLI + Telegram is all you need.
